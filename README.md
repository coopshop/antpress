# Antpress

Minimalistic React-powered static site generator.

## Demo

- [https://antpress.eggjs.app/](https://antpress.eggjs.app/)

## Usage

- install

```bash
npm install antpress
```

- build 

```bash
npx antpress bulild
npx antpress render
```

- example

```bash
https://github.com/thonatos/antpress-egg
```
