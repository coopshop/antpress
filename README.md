## Usage

- install

```bash
npm install antpress
```

- build 

```bash
npx antpress build
npx antpress render
```

- example

```bash

# download example
git clone https://github.com/thonatos/antpress-egg
cd antpress-egg

# install antpress
yarn add antpress

# build template
npx antpress build

# render docs
npx antpress render
```
